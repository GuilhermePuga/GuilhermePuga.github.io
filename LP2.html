THE NIGHT OF THE LIVING DEISI

A back-end project

Project Overview

"The Night of the Living DEISI" is a Java-based game project that simulates a zombie apocalypse scenario. The game follows a turn-based system where players navigate humans through a world infested with zombies, with the goal of survival. The project was developed in two phases, with the second phase introducing more advanced features and implementing object-oriented programming principles.

Technologies Used

This project integrates hardware and software components using modern technologies and development practices.

Java 17 Object-Oriented Programming JUnit 5 for automated testing Exception handling File I/O for game state saving/loading UML for system design GitHub for version control

Key Features

  • Entity System: Various creature types (humans and zombies) with different behaviors and attributes
  • Equipment System: Different equipment types that can be used by creatures
  • Day/Night Cycle: Game mechanics that change based on time of day
  • Safe Havens: Protected areas where humans can take refuge
  • Movement System: Turn-based movement with validity checks
  • Game State Management: Ability to save and load game states
  • Game Over Conditions: Logic to determine when the game ends
  • Survivor Tracking: Functionality to track which humans have survived

Implementation

The project follows a robust object-oriented design with proper encapsulation, inheritance, and polymorphism. The architecture is centered around the GameManager class which serves as the main controller, interacting with other components:

1. Creature Hierarchy: An abstract Creature class with specific implementations for different types (humans and zombies).

2. Equipment System: Equipment classes with different types and effects.

3. World Representation: A grid-based world with different types of squares (normal, safe haven).

4. File Handling: Custom exception handling for file operations with the InvalidFileException class.

Challenges & Solutions

Complex Entity Behavior in a Dynamic World

Challenge:

Modeling different creature behaviors efficiently while maintaining scalability and code reusability.

Solution:

Implemented a hierarchical class structure using inheritance and polymorphism to define diverse creature behaviors.

Ensuring Game State Persistence Across Sessions

Challenge:

Saving and loading game progress reliably while preventing data corruption or loss.

Solution:

Developed a custom file format and implemented serialization/deserialization logic with proper exception handling.

Designing a Safe Haven for Creatures

Challenge:

Creating a special area in the game world where creatures can be safe from monsters.

Solution:

Designed a unique type of square in the world with special properties and requirements for creatures.

Validating Movement Based on Game Rules

Challenge:

Ensuring movement rules consider creature type, equipment, environmental conditions, and world boundaries.

Solution:

Implemented a comprehensive rule system to validate movement based on multiple gameplay factors.

Achieving High Test Coverage for System Reliability

Challenge:

Guaranteeing system stability by covering all critical functionalities and edge cases in testing.

Solution:

Employed JUnit 5 to develop a robust test suite that ensures coverage of key logic and potential failure points.

                    private boolean parseBoardDimensions(Scanner scanner) throws InvalidFileException{

                    public boolean move(int xO, int yO, int xD, int yD) {
                
                        // Verificar se o destino está fora do tabuleiro
                        if (xD < 0 || xD >= this.tabuleiroColunas || yD < 0 || yD >= this.tabuleiroLinhas) {
                            return false;
                        }
                
                        // nao pode mover para uma casa que tenha um cao
                        if (caoNoDestino(xD, yD)) {
                            return false;
                        }
                
                        // Obter a criatura na posição de origem
                        String posKey = coordenadasTabuleiro(xO, yO);
                        if (conteudoTabuleiro.get(posKey) == null) {
                            return false;
                        }
                
                        String[] cria = conteudoTabuleiro.get(posKey).split(":");
                
                        if (cria.length < 2) {
                            return false;
                        }
                
                        Criatura criatura = this.criaturas.get(Integer.valueOf(cria[1]));
                        if (criatura == null) {
                            return false;
                        }
                
                        if (!fazParteDaEquipaQueJoga(criatura.getTipo())) {
                            return false;
                        }
                
                        // Verificar se a criatura pode se mover para o destino
                        if (!criatura.canMove(xD, yD, isDay())) {
                            return false;
                        }
                
                        // nao pode mover para safe haven se nao for humano
                        if (safeHavenNoDestino(xD, yD) && criatura.getTipo() != EstadoCriatura.HUMANO) {
                
                            return false;
                        }
                
                        // humano nao pode mover para cima de outro humano
                        if (humanoNoDestino(xD, yD) && criatura.getTipo() == EstadoCriatura.HUMANO) {
                            return false;
                        }
                
                        // zombie nao pode mover para cima de outro zombie
                        if (zombieNoDestino(xD, yD) && criatura.getTipo() != EstadoCriatura.HUMANO) {
                            return false;
                        }
                
                        posKey = coordenadasTabuleiro(xD, yD);
                        boolean celulaDestinoEstaVazia = conteudoTabuleiro.get(posKey) == null;
                
                
                        boolean criaturaTinhaEquipamento = criatura.getEquipamento() != null;
                
                        if (celulaDestinoEstaVazia) {
                            movimentarCriatura(criatura, xD, yD);
                        } else {
                            if (!executarAcao(criatura, xD, yD)) {
                                return false;
                            }
                        }
                
                        // se for idoso relocamos o equipamento que este tinha no tabuleiro
                        if (criatura.getTipoCriatura() == TipoCriatura.IDOSO && criaturaTinhaEquipamento) {
                            recolocarEquipamentoAposSaidaDoJogador(criatura.getEquipamento());
                            criatura.largarEquipamento();
                        }
                
                        // Incrementar contador de rodadas e modificar estado do jogo
                        this.counterRondas++;
                        modificaJogada();
                
                        mudaTurno();
                        return true;
                    }
                
                    public boolean gameIsOver(){
                        int nrRondasSemMorteOuTransformacao = counterRondas - ultimaRondaComMorteOuTransformacao;
                        if (nrRondasSemMorteOuTransformacao >= 8) {
                            int contaBala = getContaBala();
                
                            System.out.println(contaBala);
                            return true;
                        }
                
                
                
                        boolean soExistemZombiesNoTabuleiro = true;
                        boolean soExistemHumanosNoTabuleiro = true;
                
                        for (String item : conteudoTabuleiro.values()) {
                            if (item == null) {continue;}
                
                            String[] dadosItem = item.split(":");
                            String tipoItem = dadosItem[0];
                
                            // verifica se o item da celula e um 'humano'
                            if (tipoItem.equalsIgnoreCase("H")) {
                                soExistemZombiesNoTabuleiro = false;
                            }
                
                            // verifica se o item da celula e um 'zombie'
                            if (tipoItem.equalsIgnoreCase("Z")) {
                                soExistemHumanosNoTabuleiro = false;
                            }
                        }
                
                
                
                        return soExistemHumanosNoTabuleiro || soExistemZombiesNoTabuleiro;
                    }
                
                    private int getContaBala() {
                        int contaBala = 0;
                        for (Equipamento equipamento : equipamentos.values()){
                            if (equipamento.getTipo() == TipoEquipamento.PISTOLA){
                                Pistola pistola = (Pistola) equipamento;
                                contaBala += pistola.darBalas();
                            }
                        }
                
                
                        return contaBala;
                    }
                
                    public ArrayList getSurvivors() {
                        ArrayList survivors = new ArrayList<>();
                        survivors.add("Nr. de turnos terminados:");
                        survivors.add("" + counterRondas);
                
                        survivors.add("");
                        survivors.add("OS VIVOS");
                
                        for (Criatura criatura : this.criaturas.values()) {
                            if (criatura.getTipo() == EstadoCriatura.HUMANO) {
                                survivors.add(criatura.getID() + " " + criatura.getNome());
                            }
                        }
                        survivors.add("");
                        survivors.add("OS OUTROS");
                
                        for (Criatura criatura : criaturas.values()) {
                            boolean criaturaEliminada = false;
                            for (Criatura criaturaRemovida : criaturasRemovidas) {
                                if (criaturaRemovida.getID() == criatura.getID()) {
                                    criaturaEliminada = true;
                                    break;
                                }
                            }
                
                            if (criatura.getTipo() != EstadoCriatura.HUMANO && !criaturaEliminada) {
                                survivors.add(criatura.getID() + " (antigamente conhecido como " + criatura.getNome() + ")");
                            }
                        }
                
                        survivors.add("-----");
                
                        return survivors;
                    }

                    public void saveGame(File file) throws IOException {
                        try (FileWriter writer = new FileWriter(file)) {
                            // Save board dimensions
                            writer.write(tabuleiroLinhas + " " + tabuleiroColunas + "\n");
                
                            // Save current team ID
                            writer.write(equipaInicial + "\n");
                
                            // Save creatures
                            writer.write(criaturas.size() + "\n");
                            for (Map.Entry entry : criaturas.entrySet()) {
                                Criatura criatura = entry.getValue();
                
                                // Check if the creature is HUMANO (team 20) or ZOMBIE (team 10)
                                int teamId = (criatura.equipa() == 10) ? 10 : (criatura.equipa() == 20 ? 20 : 0);  // Default to 0 if the team is unknown.
                
                                // Write creature details with HUMANO as 20 and ZOMBIE as 10
                                writer.write(criatura.getID() + " : " +
                                        teamId + " : " +  // Save 10 for ZOMBIE and 20 for HUMANO
                                        criatura.getTipoCria() + " : " +
                                        criatura.getNome() + " : " +
                                        criatura.getX() + " : " +
                                        criatura.getY() + "\n");
                            }
                
                            // Save equipment
                            writer.write(equipamentos.size() + "\n");
                            for (Map.Entry entry : equipamentos.entrySet()) {
                                Equipamento equipamento = entry.getValue();
                
                                // Convert the equipment type from the enum to its numeric value (0, 1, 2, or 3)
                                int equipamentoTipo = equipamento.getTipo().ordinal(); // ordinal() returns the integer value of the enum
                
                                writer.write(equipamento.getId() + " : " +
                                        equipamentoTipo + " : " +  // Equipment types: ESCUDO = 0, ESPADA = 1, PISTOLA = 2, LIXIVIA = 3
                                        equipamento.getX() + " : " +
                                        equipamento.getY() + "\n");
                            }
                
                            // Save safe havens
                            writer.write((safeHaven.size()) + "\n");
                            for (Criatura door : safeHaven) {
                                writer.write(door.getX() + " : " + door.getY() + "\n");
                            }
                
                        } catch (IOException e) {
                            throw new IOException("Error saving game to file.", e);
                        }
                    }
                
                
                    public List getIdsInSafeHaven(){
                        ArrayList idsInSaveHaven = new ArrayList<>();

                            idsInSaveHaven.add(criatura.getID());
                        }

                    }
                
                    boolean executarAcao(Criatura autor, int destinoX, int destinoY) {
                        String posKey = coordenadasTabuleiro(destinoX, destinoY);
                        String itemDestino = conteudoTabuleiro.get(posKey);
                
                        boolean casaDestinoSafeHaven = itemDestino.equalsIgnoreCase("SH");
                
                        if (casaDestinoSafeHaven) {
                            movimentarHumanoParaSafeHaven(autor);
                            return true;
                        }
                
                        
                
                
                        String[] dadosItemDestino = itemDestino.split(":");
                        String tipoItemDestino = dadosItemDestino[0];
                        int idItemDestino = Integer.parseInt(dadosItemDestino[1]);
                
                        boolean casaDestinoTemEquipamento = tipoItemDestino.equalsIgnoreCase("E");
                
                        if (casaDestinoTemEquipamento) { // equipamento no destino
                            Equipamento equipamento = getEquipamentoPorID(idItemDestino);
                
                            // zombies destroem equipamento
                            if (autor.getTipo() != EstadoCriatura.HUMANO) {
                                autor.incrementarEquipamentosDestruidos();
                                removerEquipamentoDoTabuleiro(equipamento.getId());
                                movimentarCriatura(autor, destinoX, destinoY);
                                return true;
                            }
                
                            if (!autor.podeApanharEquipamento(equipamento.getTipo())) {return false;}
                
                
                
                
                
                            autor.apanharEquipamento(equipamento);
                            //equipamentos.remove(equipamento.getId());
                            removerEquipamentoDoTabuleiro(equipamento.getId());
                            movimentarCriatura(autor, destinoX, destinoY);
                
                        } else { // criatura no destino
                            Criatura vitima = getCriaturaPorID(idItemDestino);
                            boolean autorPodeAtacar = autor.podeAtacar(vitima, isDay());
                
                            if (!autorPodeAtacar) {
                                return false;
                            }
                
                            boolean vitimaPodeDefender = vitima.podeDefender();
                
                            // vitima defender
                            if (vitimaPodeDefender) {
                                vitima.defender();
                                return true;
                            }
                
                            // vitima foi atacada
                            if (vitima.getTipo() == EstadoCriatura.HUMANO) {
                                // vitima nao defende e e transformada em zombie
                                vitima.transformarEmZombie();
                
                                // atualiza dados tabuleiro
                                posKey = coordenadasTabuleiro(vitima.getX(), vitima.getY());
                                conteudoTabuleiro.put(posKey, vitima.getIdentificadorTabuleiro());
                
                                // atualiza counter de eliminacoes/transformacoes
                                ultimaRondaComMorteOuTransformacao = counterRondas;
                            } else {
                                // utilizar equipamento (gastar capacidade)
                                autor.getEquipamento().utilizarEquipamento();
                
                                // remove o zombie do tabuleiro
                                removerCriaturaDoTabuleiro(vitima.getID());
                
                                // movimentar o jogador para o lugar do zombie que morreu
                                movimentarCriatura(autor, destinoX, destinoY);
                
                                // atualiza counter de eliminacoes/transformacoes
                                ultimaRondaComMorteOuTransformacao = counterRondas;
                            }
                        }
                
                        // movimento valido
                        return true;
                    }
                
                    void movimentarCriatura(Criatura criatura, int xDestino, int yDestino) {
                        String posKey = coordenadasTabuleiro(criatura.getX(), criatura.getY());
                        conteudoTabuleiro.remove(posKey);
                        criatura.atualizarPosicao(xDestino, yDestino);
                        String newPos = coordenadasTabuleiro(criatura.getX(), criatura.getY());
                        conteudoTabuleiro.put(newPos, criatura.getIdentificador() + ":" + criatura.getID());
                    }
                
                    boolean removerCriaturaDoTabuleiro(int idCriatura) {
                        for (String k : conteudoTabuleiro.keySet()) {
                            String item = conteudoTabuleiro.get(k);
                            String[] dados = item.split(":");
                
                            if (dados.length < 2) {continue;}
                            int id = Integer.parseInt(dados[1]);
                
                            if (id == idCriatura) {
                                Criatura criatura = getCriaturaPorID(Integer.parseInt(dados[1]));
                                conteudoTabuleiro.remove(k);
                                criaturasRemovidas.add(criatura);
                                return true;
                            }
                        }
                        return false;
                    }
                
                    boolean removerEquipamentoDoTabuleiro(int idEquipamento) {
                
                        for (String k : conteudoTabuleiro.keySet()) {
                            String item = conteudoTabuleiro.get(k);
                            String[] dados = item.split(":");
                            String tipoItem = dados[0];
                
                            if (dados.length < 2) {continue;}
                            int idItem = Integer.parseInt(dados[1]);
                
                            if (tipoItem.equalsIgnoreCase("E") && idItem == idEquipamento) {
                                conteudoTabuleiro.remove(k);
                                return true;
                            }
                        }
                        return false;
                    }
                
                    Criatura getCriaturaPorID(int idCriatura) {
                        for (Integer id : criaturas.keySet()) {
                            if (id == idCriatura) {return criaturas.get(id);}
                        }
                        return null;
                    }
                
                    Equipamento getEquipamentoPorID(int idEquipamento) {
                        for (Integer id : equipamentos.keySet()) {
                            if (id == idEquipamento) { return equipamentos.get(id);}
                        }
                        return null;
                    }
                
                    boolean fazParteDaEquipaQueJoga(EstadoCriatura tipoCriatura) {
                
                        // Verefica se é um humano e se é o seu turno
                        if(tipoCriatura == EstadoCriatura.HUMANO && vivos){
                            return true;
                        }
                        // Verefica se é um zombie/Vampiro e verefica se é o seu turno
                        else if((tipoCriatura == EstadoCriatura.ZOMBIE || tipoCriatura == EstadoCriatura.ZOMBIE_TRANSFORMADO) && !vivos){
                            return true;
                        }
                
                        // Caso contrario retorna false
                        return false;
                
                    }
                
                    // Faz Switch aos turnos
                    void mudaTurno(){
                        vivos = !vivos;
                    }
                
                    boolean caoNoDestino(int destX, int destY) {
                        String posKey = coordenadasTabuleiro(destX, destY);
                
                        String celulaDestino = this.conteudoTabuleiro.get(posKey);
                        if (celulaDestino == null) {
                            return false;

                        int id = Integer.parseInt(dados[1]);
                
                        Criatura criatura = getCriaturaPorID(id);
                        if (criatura == null) {
                            return false;
                        }
                
                        return criatura.getTipoCriatura() == TipoCriatura.CAO;
                    }

                package pt.ulusofona.lp2.thenightofthelivingdeisi;

                import java.awt.*;

                enum EstadoCriatura {
                    HUMANO,
                    ZOMBIE,
                    ZOMBIE_TRANSFORMADO
                }
                
                public abstract class Criatura {
                    private int id;
                    private EstadoCriatura tipo;
                    private String nome;
                    private Image img;
                    private Equipamento equipamento;
                    private ArrayList idDeEquipamentosApanhados = new ArrayList<>();
                    private int contadorEquipamentosDestruidos;
                    private int x, y ;
                    private int idEquipa;
                    private int tipoCriatura;
                
                    public Criatura(int id, int idEquipa, String nome, int x, int y, Image img) {
                        this.id = id;
                        this.nome = nome;
                        this.x = x;
                        this.y = y;
                        this.img = img;
                        this.idEquipa = idEquipa;
                
                        if (idEquipa == 10) {
                            this.tipo = EstadoCriatura.ZOMBIE;
                        } else {
                            this.tipo = EstadoCriatura.HUMANO;
                        }
                    }
                
                    int getTipoCria(){
                        return 0;
                    }
                
                    int equipa(){
                        return idEquipa;
                    }
                
                    int getID() {
                        return this.id;
                    }
                
                    String getNome() {
                        return this.nome;
                    }
                
                    int getX() {
                        return this.x;
                    }
                
                    int getY() {
                        return this.y;
                    }
                
                    Equipamento getEquipamento() {
                        return this.equipamento;
                    }
                
                    void atualizarPosicao(int x, int y) {
                        this.x = x;
                        this.y = y;
                
                        // nao atualizar a posicao do equipamento quando e idoso
                        if (this.getTipoCriatura() != TipoCriatura.IDOSO) {
                            if (this.equipamento != null) {
                                this.equipamento.atualizarPosicao(x, y);
                            }
                        }
                    }
                
                    EstadoCriatura getTipo() {
                        return this.tipo;
                    }
                
                    public abstract TipoCriatura getTipoCriatura();
                
                    String getTipoString() {
                        return switch (this.tipo) {
                            case HUMANO -> "Humano";
                            case ZOMBIE -> "Zombie";
                            default -> "Zombie (Transformado)";
                        };
                    }
                
                    char getIdentificador() {
                        if (this.tipo == EstadoCriatura.HUMANO) {
                            return 'H';
                        }
                        return 'Z';
                    }
                
                    abstract String getTipoCriaturaString();
                
                    abstract boolean canMove(int destinoX, int destinoY, boolean isDay);
                
                    boolean temEquipamentoUtilizavel() {
                        if (this.equipamento == null){ return false;}
                        return equipamento.equipamentoUtilizavel();
                    }
                
                    boolean temEquipamentoOfensivo() {
                        if (this.equipamento == null) {
                            return false;
                        }
                        return this.equipamento.getUtilidade() == UtilidadeEquipamento.OFENSIVO;
                    }
                
                    boolean podeApanharEquipamento(TipoEquipamento tipoEquipamento) {
                        if (this.getTipo() == EstadoCriatura.HUMANO) {
                            return this.podeApanharEquipamentoEnquantoHumano(tipoEquipamento);
                        }
                        return false;
                    }
                
                    abstract boolean podeApanharEquipamentoEnquantoHumano(TipoEquipamento tipoEquipamento);
                
                    boolean apanharEquipamento(Equipamento equipamento) {
                        if (podeApanharEquipamentoEnquantoHumano(equipamento.getTipo())) {
                            this.idDeEquipamentosApanhados.add(equipamento.getId());
                            this.equipamento = equipamento;
                            return true;
                        }
                        return false;
                    }
                
                    int totalEquipamentos() {
                        return this.idDeEquipamentosApanhados.size();
                    }
                
                    boolean temEquipamentoTipo(int idTipoEquipamento) {
                        if (this.equipamento == null) {return false;}
                
                        if (idTipoEquipamento == 0 && this.equipamento.getTipo() == TipoEquipamento.ESCUDO) {return true;}
                        if (idTipoEquipamento == 1 && this.equipamento.getTipo() == TipoEquipamento.ESPADA) {return true;}
                        if (idTipoEquipamento == 2 && this.equipamento.getTipo() == TipoEquipamento.PISTOLA) {return true;}
                        if (idTipoEquipamento == 3 && this.equipamento.getTipo() == TipoEquipamento.LIXIVIA) {return true;}
                
                        return false;
                    }
                
                    boolean podeAtacar(Criatura vitima, boolean isDay) {
                        if (this.getTipo() == EstadoCriatura.HUMANO) {
                            if (!this.temEquipamentoOfensivo() || !this.temEquipamentoUtilizavel()) {
                                return false;
                            }
                
                            if (vitima.getTipo() == EstadoCriatura.HUMANO) {
                                return false;
                            }
                
                            return podeAtacarEnquantoHumano(vitima, isDay);
                        } else {
                            return podeAtacarEnquantoZombie(vitima, isDay);
                        }
                    }
                
                    boolean podeDefender() {
                        return this.getTipo() == EstadoCriatura.HUMANO && temEquipamentoUtilizavel();
                    }
                
                    abstract boolean podeAtacarEnquantoHumano(Criatura vitima, boolean isDay);
                
                    abstract boolean podeAtacarEnquantoZombie(Criatura vitima, boolean isDay);
                
                    void defender() {
                        this.equipamento.utilizarEquipamento();
                    }
                
                    void transformarEmZombie() {
                        this.tipo = EstadoCriatura.ZOMBIE_TRANSFORMADO;
                
                        // equipamentos apanhados passam a destruidos
                        this.contadorEquipamentosDestruidos = this.totalEquipamentos();
                        // deixa de ter equipamento
                        this.equipamento = null;
                    }
                
                    String getIdentificadorTabuleiro() {
                        String tipo = "";
                        if (this.tipo == EstadoCriatura.HUMANO) {
                            tipo = "H";
                        } else {
                            tipo = "Z";
                        }
                        return tipo + ":" + this.id;
                    }
                
                    void incrementarEquipamentosDestruidos() {
                        this.contadorEquipamentosDestruidos++;
                    }
                
                    int totalEquipamentosDestruidos() {
                        return this.contadorEquipamentosDestruidos;
                    }
                
                    ArrayList getIdDeEquipamentosApanhados() {
                        return this.idDeEquipamentosApanhados;
                    }
                
                    void largarEquipamento() {
                        this.equipamento = null;
                    }
                
                }
