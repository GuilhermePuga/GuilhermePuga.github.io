OMICRON NETWORK INFRASTRUCTURE

Corporate Multi-Site Network Implementation using Cisco Packet Tracer

Project Overview

This project describes the comprehensive implementation and analysis of the network infrastructure for Omicron Lda., developed as an individual capstone project for the Computer Networks course. The project demonstrates advanced networking concepts through the creation of a sophisticated corporate network with headquarters in Coimbra and branches in Porto, Tavira, Viseu, and Ericeira, utilizing the Cisco Packet Tracer simulator for realistic network modeling.

Project Achievements

  • Successfully designed and implemented a scalable multi-site corporate network topology
  • Configured advanced VLAN segmentation for enhanced security and network organization
  • Deployed essential network services including DHCP, DNS, and Web services
  • Established robust inter-site connectivity with optimal routing protocols
  • Implemented centralized Internet access with proper security measures
  • Conducted comprehensive testing and performance analysis

Network Architecture

Topology Design

The network architecture features a hierarchical design with the main headquarters in Coimbra serving as the central hub (3 floors) and four strategically located branch offices (Porto, Tavira, Viseu, Ericeira) each with 1 floor. The design ensures high-speed connectivity with 1 Gbps inter-site links and 100 Mbps internal connections, while Internet access is centralized through Coimbra with a dedicated 100 Mbps connection.
